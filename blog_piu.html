Hardware & Programs I Use

This blog entry explores all of the computer hardware and sofware I prefer and use on a daily basis. Most software is command-line-based and for the pragmatic man only. I made the switch to the Linux distro 'Manjaro' in the latter end of 2020, using a Tiling Window Manager. I've also created a similar setup on my Mac which allows a more non-MacOS method of communicating with MacOS. This is simply a far more effcient and easier way to operate machines. Specifically, with regards to the Mac, the best thing I've ever done is hide that goddamn dock and menu bar!

Hardware

  • Daily Driver: Blackberry Key2
  • VimPad: Lenovo ThinkPad T420
  • NeovimPad: Lenovo ThinkPad X220
  • MemePad: IBM Thinkpad T40
  • EmacsBook: MacBook Pro 2015
  • Keyboard: IBM Model M 1991

Software

  • Operating System: MacOS X Big Sur, Manjaro
  • Tiling Window Manager: Amethyst (MacOS), i3-gaps (Manjaro)
  • Terminal: Terminal (MacOS), st (Simple Terminal) (Manjaro)
  • Shell: zsh
  • Text Editor: Vim, Neovim
  • File Manager: ranger
  • Mail Client: mutt
  • Music/Audio Player: cmus
  • Video Player: mpv
  • Writing Documents: Markdown, R Markdown, LaTeX
  • Compliling Documents: pandoc, pdflatex
  • Comma-Separated-Value Spreadsheets: SC-IM
  • Presentations: Beamer
  • PDF Viewer: MuPDF, zathura
  • Image Viewer: dxiv
  • Image Modifier: GIMP
